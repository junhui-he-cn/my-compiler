# Map `merge` Implementation Plan

**Goal:** Add `merge(left, right)` and `left.merge(right)` as a fresh,
insertion-ordered, shallow map composition helper.

**Architecture:** Register `merge` as a shadowable native function, reuse
`native_call` for function and member forms, merge known map component types in
the type checker, and reuse the Rust VM map constructor's ordered duplicate-key
handling.
